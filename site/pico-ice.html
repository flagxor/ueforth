Install

  • Connect your pico-ice to USB.
  • Connect the USB_BOOT pin to ground.
  • Press reset.
  • Copy the ueforth-pico-ice.uf2 to the mounted drive.


See detailed instructions for MCU programming

Use

Initially ÂµEforth can be interacted with over a serial port (over USB). One popular option is to use the serial port option on PuTTY. Be sure to config the serial port to: baud rate = 115200, data bits = 8, stop bits = 1, and parity = N.
